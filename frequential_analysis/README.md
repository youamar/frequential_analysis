## How to build the program

Once you downloaded the project, please move inside the project's main folder named : FrequentialAnalysis
when you are in, you can navigate inside the directories : src/main/java.

Now that you are in the "java" folder you can type the following command to build the program :
```sh
javac Cryptanalysis.java
```
If the command didn't work please type the following command before re-trying the same previous command :
```sh
sudo apt install build-essential default-jdk default-jre 
```
Now that the project compiled you should have a file named : Cryptanalysis.class
you can now run the program by typing the following command :
```sh
java Cryptanalysis
```
