# Frequential Analysis
A Java application that cracks ciphers using frequential analysis, helping users decrypt secret messages.
## Screenshot
![App Screenshot](https://i.ibb.co/frrsSGY/8.png)
## Run Locally
Clone the project
```bash
  git clone https://github.com/youamar/frequential_analysis.git
```
Go to the project directory (frequential_analysis) and move inside the java folder
```bash
  src/main/java
```
Now run the program using the file called "Cryptanalysis.java"
